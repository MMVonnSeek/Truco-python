## Sobre o Projeto
Este repositório foi desenvolvido para um desafio de Python aplicado aos meus alunos no SENAI-DF. O projeto consiste em um jogo completo de Truco Mineiro, utilizando Tkinter para criar a interface gráfica e implementando toda a lógica de hierarquia de cartas, rodadas e pontuação. A atividade também reforça boas práticas de versionamento e publicação no GitHub.

---

## Objetivos Educacionais
O código demonstra, na prática, conceitos fundamentais valorizados por recrutadores:

### Interface gráfica com Tkinter
Criação de janelas, botões, labels, eventos e manipulação visual.

### Estruturação de um jogo real
Gerenciamento de rodadas, estados, pontuação e fluxo da partida.

### Lógica aplicada
Implementação completa da hierarquia tradicional do Truco Mineiro com manilhas fixas.

### Funções, classes e modularidade
Organização do código com funções e uma classe principal que controla todo o jogo.

### Programação orientada a eventos
Interações dinâmicas entre jogador e computador, atualizadas em tempo real.

## Conteúdo do Projeto
O repositório inclui:

- Interface gráfica em Tkinter

- Sistema de comparação de cartas  
- Exibição de resultados por `messagebox`
